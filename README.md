# LifeLink API

**LifeLink API** is a backend service built with Spring Boot that provides RESTful endpoints for managing LifeLink patient data and interactions.


## Table of Contents

- [Used Technologies](#used-technologies)
- [Features](#features)
- [API Documentation](#api-documentation)
- [Support](#support)


## Used Technologies

- **Java 21**
- **Spring Boot 3.5.6**
- **Gradle 8.14.3**


## Features

- RESTful API endpoints for managing patient data such as:
  - Patient registration
  - Patient data retrieval and updates (allergies, chronic diseases, medicines, vaccinations, medical checkups, medical diagnoses and medical procedures)
  - Appointment scheduling
- Data validation and error handling
- Integration with a relational database (MySQL by default)
- NFC tag support for patient information retrieval
- Secure authentication and authorization
